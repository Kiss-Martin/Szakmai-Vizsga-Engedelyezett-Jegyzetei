# Jegyzet Repo

Szakmai Vizsga Engedélyezett Jegyzetei

## Funkciók

- **Szekciók**: C# Console App, WPF, HTML, Frontend (Angular, React), Backend (Laravel, ASP.NET, Next.js)
- **Keresés**: Szűrés a jegyzetek között
- **Másolás**: Egy kattintással másolhatod a tartalmat
- **Téma**: Világos/sötét mód váltó

## Használat

1. Nyisd meg az `index.html` fájlt böngészőben
2. Választ szekciót a legördülő menüből (vagy hagyd az "Minden szekció" opciót)
3. Írd be a keresőkifejezést a keresősávba
4. Kattints a "Másolás" gombra egy box tartalmának másolásához
5. Kattints a 🌙/☀️ gombra a téma váltásához
